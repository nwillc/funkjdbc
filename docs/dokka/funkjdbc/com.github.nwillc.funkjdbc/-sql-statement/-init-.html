SqlStatement(sql: String, binder: Binder? = null)

A SQL string, with a binding block in needed to help with JDBC PreparedStatement. The sql String excepts JDBC '?' value replacement syntax, and the binding block allows you to bind values to those '?'s on a given PreparedStatement.

In the simplest case:

val sql = SqlStatement("SELECT * FROM WORDS")

An example with bindings:

val sql = SqlStatement("SELECT * FROM WORDS WHERE COUNT < ?") {
  it.setInt(1, someValue)
}
