fun <T, R> Connection.query(sql: String, extractor: Extractor<T>, resultsProcessor: ResultsProcessor<T, R>): R

Execute a SQL query on a JDBC Connection. The SQL is expected to be a query and therefore returns a result set that the extractor can extract type T from the rows, and the resultsProcessor can then process them.

Parameters

sql - A simple SQL query.

extractor - A function to extract type T from the rows.

resultsProcessor - A function to process rows after extraction.
